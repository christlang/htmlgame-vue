Dieses Spiel ist ein Demo-Projekt für die Programmierung mit Vue.js; es wird ausführlich erklärt in einem Tutorial in c't Ausgabe 12/18. Das gleiche Projekt wurde bereits mit React umgesetzt (Demo, Code), um die beiden Bibliotheken vergleichen zu können. Die Vue.js-Version hat allerdings einige zusätzliche Features.

Datengrundlage ist die W3C-Empfehlung HTML 5.1-2 (Oktober 2017); als experimentell hinzugefügt wurden die in WHATWG HTML (Stand: Mai 2018) zusätzlich verzeichneten Elemente.
